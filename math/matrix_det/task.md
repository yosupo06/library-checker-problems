## @{keyword.statement}
$N \times N$ 正方行列 $a_{ij}$ が与えられます。行列式をmod 998244353で求めてください。

## @{keyword.constraints}

- $1 \leq N \leq 500$
- $0 \leq a_{ij} < 998244353$

## @{keyword.input}

```
$N$
$a_{11}$ $a_{12}$ ... $a_{1N}$
$a_{21}$ $a_{22}$ ... $a_{2N}$
:
$a_{N1}$ $a_{N2}$ ... $a_{NN}$
```
